## Supported intent types

Currently, only one intent type is supported:

### Lend

Deposit funds into DeFi lending protocols (Aave, Compound, Morpho):

```json
{
  "action": "lend",
  "amount": "1000",
  "asset": "USDC",
  "chain": "ethereum-sepolia"
}
```

## Wallet limits

Subscription plans determine how many wallets you can create:

| Plan | Wallet Limit |
|------|--------------|
| Free | 100 wallets |
| Pro | 10,000 wallets |
| Enterprise | Unlimited |

<Info>
  Wallet limits apply to the number of wallets you can create, not the number of intents you can execute or the transaction volume.
</Info>

## Rate limits

API rate limits based on your plan:

| Plan | Requests per minute |
|------|---------------------|
| Free | 100 |
| Pro | 1,000 |
| Enterprise | Custom |

## Migration path

When mainnet launches, we'll provide:

- Migration guides for moving from testnet to mainnet
- Tools to help transition wallet addresses
- Support for transferring testnet data

<Note>
  **No automatic migration**: Testnet wallets and intents will not automatically migrate to mainnet. You'll need to create new wallets and operations on mainnet.
</Note>
